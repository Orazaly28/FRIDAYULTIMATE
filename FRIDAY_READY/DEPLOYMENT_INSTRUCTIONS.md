# 🚀 F.R.I.D.A.Y. APK — Автоматическая сборка через GitHub Actions

## ШАГ 1: Создать публичный репозиторий

1. Перейти на https://github.com/new
2. Название репозитория: `FRIDAY-Ultimate`
3. Описание: `F.R.I.D.A.Y. AI Assistant - Ultimate Edition`
4. Видимость: **Public** (обязательно!)
5. ✅ Add a README file
6. Нажать **Create repository**

## ШАГ 2: Загрузить файлы проекта

**Метод A: Через веб-интерфейс (рекомендуется)**
1. На странице репозитория нажать **Add file → Upload files**
2. Скачать проект: https://www.genspark.ai/sandbox/file-viewer?project_id=612fbf4b-7d55-4d38-b66f-1dfbc0ada4be&path=%2Fmnt%2Fuser-data%2Foutputs%2FFRIDAY_MOBILE_FIXED.zip
3. Распаковать архив на компьютере
4. Перетащить ВСЕ файлы из папки FRIDAY_MOBILE в окно браузера
5. Commit message: `Initial commit - FRIDAY Ultimate`
6. Нажать **Commit changes**

**Метод B: Через GitHub Mobile (на телефоне)**
1. Установить приложение **GitHub** из Google Play
2. Войти в аккаунт
3. Открыть репозиторий `FRIDAY-Ultimate`
4. Меню → **Upload files**
5. Выбрать файлы из скачанного архива
6. Commit

## ШАГ 3: Запустить автоматическую сборку

1. В репозитории перейти во вкладку **Actions**
2. Если появится "Workflows aren't being run" → нажать **I understand my workflows, go ahead and enable them**
3. Слева выбрать **Build F.R.I.D.A.Y. APK**
4. Справа нажать **Run workflow** → **Run workflow**
5. Начнётся сборка (8-12 минут)

## ШАГ 4: Скачать готовый APK

1. Дождаться зелёной галочки ✅ рядом с запуском
2. Кликнуть на название запуска (например, "Build F.R.I.D.A.Y. APK #1")
3. Прокрутить вниз до секции **Artifacts**
4. Скачать **FRIDAY-APK** (~15-20 MB)
5. Распаковать скачанный ZIP
6. Внутри будет `app-debug.apk`

## ШАГ 5: Установить на телефон

1. Перенести `app-debug.apk` на телефон (любым способом)
2. Открыть файл
3. Разрешить установку из неизвестных источников (если попросит)
4. Нажать **Установить**
5. Открыть приложение **F.R.I.D.A.Y.**
6. Разрешить все запрошенные права (микрофон, контакты, звонки, accessibility)

## ✅ ГОТОВО!

После установки F.R.I.D.A.Y. автоматически запустится. Скажите:
- **"Привет, Пятница"** — активация
- **"Активируй Веронику"** — режим учёбы
- **"Время расслабиться"** — режим отдыха
- **"Гаси свет"** — ночной режим

## ⚠️ Возможные проблемы

**"Actions не запускаются"**
→ Убедитесь, что репозиторий **Public**, а не Private

**"Build failed"**
→ Откройте лог сборки, найдите строку с ERROR, пришлите скриншот

**"APK не устанавливается"**
→ Проверьте версию Android (нужен 7.0+)
→ Отключите Play Protect на время установки

**"Приложение вылетает"**
→ Убедитесь, что дали ВСЕ разрешения в настройках приложения
