# 🤖 F.R.I.D.A.Y. Ultimate Assistant

Продвинутый голосовой AI-ассистент в стиле Stark Industries.

## 📱 Автоматическая сборка APK

Этот проект настроен для автоматической сборки через GitHub Actions.

### Как получить готовый APK:

1. Форкните этот репозиторий
2. Перейдите в раздел **Actions**
3. Нажмите **"Build F.R.I.D.A.Y. APK"**
4. Нажмите **"Run workflow"** → **"Run workflow"**
5. Дождитесь завершения (8-12 минут)
6. Скачайте APK из раздела **Artifacts**

## ✨ Возможности

- 🎤 Голосовое управление
- 👥 26 VIP-контактов
- 📞 Автоответ на звонки
- 📸 Анализ фотографий (OCR, объекты, эмоции)
- 🎵 Управление музыкой
- 📅 Календарь и напоминания
- 🌐 Веб-поиск голосом
- 🎨 Создание контента
- 💾 Резервное копирование
- 🇰🇿 Поддержка казахского языка

## 🛠️ Режимы работы

- **ВЕРОНИКА** — максимальная концентрация (учёба)
- **ВЕЧЕРИНКА** — отдых и развлечения
- **ТИШИНА** — ночной режим

## 📦 Установка

1. Скачайте APK из Releases или Actions
2. Установите на устройство Android 7.0+
3. Предоставьте необходимые разрешения
4. Активируйте Accessibility Service
