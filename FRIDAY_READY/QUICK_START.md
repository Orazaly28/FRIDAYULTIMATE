# 🚀 БЫСТРЫЙ СТАРТ

## Вы форкнули этот репозиторий!

Теперь запустите автоматическую сборку APK:

### Шаг 1: Включите GitHub Actions
1. Перейдите во вкладку **Actions** (верхнее меню)
2. Нажмите зелёную кнопку **"I understand my workflows, go ahead and enable them"**

### Шаг 2: Запустите сборку
1. Выберите **"Build F.R.I.D.A.Y. APK"** (слева)
2. Справа нажмите **"Run workflow"** → **"Run workflow"**
3. Дождитесь завершения (8-12 минут)

### Шаг 3: Скачайте APK
1. Кликните на завершённую сборку (зелёная галочка ✅)
2. Прокрутите вниз → **"Artifacts"**
3. Скачайте **"FRIDAY-APK"** (15-20 MB)
4. Распакуйте ZIP → установите `app-debug.apk` на телефон
